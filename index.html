Come creare un template per i post di Mario Rossi

La prima cosa da fare è suddividere in blocchi logici la pagina, identificando le parti comuni da applicare a tutte le pagine

    Le varie arre saranno quindi suddivise in:
  • Contenuti di intestazione del tag <header>
  • Contenuti centrali variabili del tag <main>
  • Contenuti di fine pagina comuni, da inserire nel tag <footer>
  • Formattazione del codice
    Il testo dell'articolo dovrà essere formattato per identificare tutte le parti da selezionare con il CSS. Perciò i passi saranno:
  • Progettare il markup assegnando ad ogni specifico testo il tag appropriato
  • Scrivere l'HTML dai tag esterni a quelli interni e dall'alto verso il basso
  • Applicare il CSS
  • L'applicazione di stili CSS + crescita negli fino a raggiungere la diffusione pressoché totale.
